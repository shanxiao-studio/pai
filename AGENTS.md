# AGENTS Instruction

PAI is a new era of software project management, powered by LLM agents.

Linear + LLM Chat + Agent Orchestration = PAI

## Reference

- tech stack: Electron, Vite, TypeScript, React, Shadcn UI, Lucide Icons
- [project architecture](docs/pai-architecture.drawio)
- [project orchestration inspiration by OpenAI's Symphony](https://github.com/openai/symphony/blob/main/SPEC.md)

## Development Instruction

- UI 中能抽取成通用界面组件的功能，优先抽取成组件；界面形式要统一。
- 为新增或修改的代码（UI 除外）增加单元测试，必要时可修改测试代码。
